Here we work on the Hospital Cost Report Information System (HCRIS) data. We analyze the distribution of total charges and estimated prices over the years, along with ATEs via instrumental variables. Specifically, we estimated the effects of HRRP/VBP penalties on hospital prices, using prior Medicare discharges (e.g., exposure to the program) as an instrument. We focused only on a cross-sectional analysis of price changes from 2011 to 2014 as a function of 2012 penalties and average pre-2012 Medicare discharges (2009-2011), where we defined penalty as the net penalty under both HRRP and VBP.
